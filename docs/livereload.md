# Live reload

https://github.com/omeid/go-livereload

## Download
```
go get -u github.com/omeid/go-livereload/cmd/livereload
```

## Install
```
go install -i github.com/omeid/go-livereload/cmd/livereload
```

## Use
```
livereload . -serve "static" -strip "static"
```
